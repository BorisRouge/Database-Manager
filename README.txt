ИНСТРУКЦИЯ:

Предполагается наличие установленного docker.

Запускается через команду docker compose up (или аналогичную команду в вашем терминале). 

Веб-страница должна быть по адресу http://0.0.0.0:8000/overview/. (ip сервера выводится в терминал при запуске процесса canalservice-web).

Уведомления в телеграм приходят от бота https://t.me/CanalServiceTest_bot. Мониторинг запускается командой /start, интервал уведомлений 1 сутки.

Частота обновления БД установлена на 10 секунд.
